# AuthLX

AuthLX builds modern tools and services for developers.

## Open Source

While the AuthLX platform and its production infrastructure are proprietary, we open source selected projects to help developers integrate with our services. These include:

* Official SDKs
* API clients
* Example applications
* Developer tools
* Documentation and sample code

## Community

Questions, feedback, and feature suggestions are welcome through GitHub Discussions.
